Ecuación de difusión (FTCS)

Alan Turing propuso el siguiente modelo para la morfogénesis (la formación de estructuras espaciales en el desarrollo embrionario): \begin{equation} \frac{\partial u}{\partial t} = \mu \frac{\partial ^ 2 u}{\partial x ^ 2} + g_1(u, v) \end{equation} \begin{equation} \frac{\partial v}{\partial t} = \nu \frac{\partial ^ 2 v}{\partial x ^ 2} + g_2(u, v) \end{equation} con las siguientes interacciones entre ambas substancias: \begin{equation} g_1(u, v) = \frac{1}{32}(-7 u^2 - 50 u v + 57) \end{equation} \begin{equation} g_2(u, v) = \frac{1}{32}(7 u^2 + 50 u v - 2v - 55) \end{equation}

Para resolver el sistema de ecuaciones se utilizará una discretizacion en el tiempo y en el espacio, con resolución $\Delta t = 0.001$ y $\Delta x = 0.01$. La derivada temporal se calculará adelantada y la segunda derivada espacial centrada y con condiciones de borde periódicas. Es por eso que el método se le llama FTCS (Forward in Time, Centered in Space). Si se toma el supraíndice $n$ para los pasos del tiempo y el subíndice $j$ para la coordenada espacial: \begin{equation} u_j^{n + 1} = u_j^n + D \frac{\Delta t}{\Delta x^2}(u_{j+1}^n-2 u_{j}^n+u_{j-1}^n). \end{equation}

Veamos los resultados para diferentes coeficientes de difusión, siempre tomando $\mu = D$ y $\nu = D/2$. El estado inicial del sistema es el mismo para cada uno de los valores de $D$ utilizados, y se tomaron como valores aleatorios en un rango de $\pm 0.15$ alrededor de 1 para cada una de las funciones $u$ y $v$.

$D = 0.01$

Figura 1: Evolución en el tiempo del sistema con $D = 0.01$. A partir de $t=20$ el sistema alcanza el estado estacionario.

Con $D = 0.01$ el sistema evoluciona hacia el estado estacionario con longitud de onda $\lambda = 1$. Como se puede ver en la figura 1, entre $t=20$ y $t=50$ el sistema no cambia mucho, por lo que ya se alcanzó el estado estacionario.

$D = 0.0025$

Figura 2: Evolución en el tiempo del sistema con $D = 0.0025$. A partir de $t=200$ el sistema alcanza el estado estacionario.

Con $D = 0.0025$ el sistema evoluciona hacia el estado estacionario con longitud de onda $\lambda = 0.5$. Entre $t=200$ y $t=300$, como muestra la figura 2, el sistema no cambia mucho, el estacionario se alcanza entonces, alrededor de $t =200$.

$D = 0.0015$

Figura 3: Evolución en el tiempo del sistema con $D = 0.0015$. A partir de $t=200$ el sistema alcanza el estado estacionario.

Con $D = 0.0015$ el sistema evoluciona hacia el estado estacionario con longitud de onda $\lambda \simeq 0.33$. Entre $t=200$ y $t=300$, como muestra la figura 3, el sistema no cambia mucho, el estacionario se alcanza entonces, alrededor de $t =200$.

$D = 0.00075$

Figura 4: Evolución en el tiempo del sistema con $D = 0.00075$. A partir de $t=200$ el sistema alcanza el estado estacionario.

Con $D = 0.00075$ el sistema evoluciona hacia el estado estacionario con longitud de onda $\lambda = 0.25$. Entre $t=200$ y $t=300$, como muestra la figura 4, el sistema no cambia mucho, el estacionario se alcanza entonces, alrededor de $t =200$.

$D = 0.0005$

Figura 5: Evolución en el tiempo del sistema con $D = 0.0005$. El programa lanzó una excapción por nan en $t = 272.938$.

Con $D = 0.0005$ el sistema parece evolucionar hacia el estado estacionario con longitud de onda $\lambda \simeq 0.20$. En $t = 272.938$ el programa que implmenté lanza una excepción por nan.

Figura 6: Longitud de onda del estado estacionario en función del coeficiente de difusión.

La longitud de onda del estado estacionario tiene una dependencia del coeficientes de difusión $D$ utilizado como se muestra en la figura 6.
