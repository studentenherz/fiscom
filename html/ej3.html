Runge-Kutta de órden 4 (RK4)

Los métodos de Runge-Kutta son una familia de métodos numéricos para resolver ecuaciones diferenciales con distcretización del tiempo de modo similar al Método de Euler. Se realizan aproximaciones sucesivas apuntando a aumentar el orden del error del cálculo. Un versión muy utilizada de estos métodos es el Runge-Kutta de orden 4 (RK4).

Se quiere resolver el sistema \begin{equation} \frac{d \boldsymbol{x}(t)}{dt} = f(\boldsymbol{x}, t); \,\,\,\,\,\,\,\,\,\,\,\, f(t_0) = \boldsymbol{x}_0. \label{eq:prob} \end{equation} Si se divide el tiempo en pasos finitos $t \to \{t_0, t_0+h, ..., t_0+ Nh\}$ y llamamos $t_j = t_0 + j h$ y $\boldsymbol{x}_j = \boldsymbol{x}(t_j)$, con $j = 0, 1, ..., N$, el método RK4 consiste en los siguientes pasos: \begin{align*} \boldsymbol{k}_1 &= h \times f(\boldsymbol{x}_j, t_j) \\ \boldsymbol{k}_2 &= h \times f\left(\boldsymbol{x}_j + \frac{\boldsymbol{k}_1}{2}, t_j + \frac{h}{2} \right)\\ \boldsymbol{k}_3 &= h \times f\left(\boldsymbol{x}_j + \frac{\boldsymbol{k}_2}{2}, t_j + \frac{h}{2} \right)\\ \boldsymbol{k}_4 &= h \times f\left(\boldsymbol{x}_j + \boldsymbol{k}_3, t_j + h \right), \end{align*} luego, la solución para el próximo paso es \begin{equation} \boldsymbol{x}_{j+1} = \boldsymbol{x}_j + \frac{1}{6}(\boldsymbol{k}_1 + 2\boldsymbol{k}_2 + 2\boldsymbol{k}_3 + \boldsymbol{k}_4 ) + \boldsymbol{o}(h^5) \,\,\,\,\,\,\,\,\,\,\,\, t_{j+1} = t_j + h \end{equation}

Un sistema no lineal

Utilicemos el método para resolver un problema. Se tiene el siguiente sistema de ecuaciones que describe la evolución de las concentraciones de dos especies en una reacción química \begin{align} \frac{dx}{dt} &= A - (B + 1)x + x^2y \\ \frac{dy}{dt} &= Bx - x^2y \label{eq:sys} \end{align} donde $A$ y $B$ son parámetros que fijan el comportamiento del sistema.

Caso estable

Veamos qué se obtiene para $A = 1$ y $B = 1$. Con condición inicial $(1.45, 1.45)$ y varios valores de $h$ se obtiene lo que muestra la siguiente figura

Aplicación del método de Runge-Kutta de orden 4 el sistema de ecuaciones \eqref{eq:sys}. Se puede notar cómo cuánto más pequeño es el calor de $h$ mejor es la aproximación.

El algoritmo es $o(h^5)$ en cada paso, y el número de pasos totales depende del paso como $1/h$ por lo que el algoritmo en total es de orden 4. Disminuir en un orden de magnitud $h$ implica una mejora en la precisión de $10^4$. Se puede notar que incluso con $h = 0.5$ la aproximación en bastante (lo que sea que eso signifique) buena.

Para esta elección de parámetros el sistema tiene un equilibrio estable en $(1, 1)$ y las todas soluciones convergen a ese punto. Incluso con $h = 1$, este sistema es estable y el método se comporta relativamente bien.

Se puede ver como todas las soluciones convergen a punto de equilibrio, incluso con paso $h = 1$.

Caso de ciclo límite

Ahora veamos qué pasa si tomamos $A = 1$ y $B = 3$. En este caso utilizaremos un conservador $h = 0.1$.

Solución utilizando RK4 con $A = 1$, $B = 3$ y un paso $h = 0.1$. El sistema es inestable y las soluciones tienden a un ciclo límite que envuelve el punto fijo en $(1, 3)$. La leyenda dice dónde fue el punto inicial.

Este sistema es inestable y las soluciones tienden a un ciclo límite que envuelve su punto fijo. Se puede ver cómo en la parte de abajo de la curva el método no se comporta bien. En estas regiones donde hay cambios muy grandes sería necesario que el $h$ fuese menor para lograr un buen resultado. De lo calculado con condiciones iniciales $(3.2, 3.6)$ se ve más claramente que el método no lo maneja bien y eventualmente, con un $h$ suficientemente grande, o alguna condición inicial particular, puede divergir. Para ver esto se probó llevar al límite al RK4 con condiciones iniciales fijas. 

try

Esto deja claro que RK4 es bueno siempre y cuando el sistema sea bien comportado. En caso de que, estemos en condiciones de inestabilidad se necesitará un $h$ bien pequeño, mientras para un sistema estable un $h$ relativamente grande puede hacer bien el trabajo.
